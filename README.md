# Friday - Advanced AI Assistant

Friday is an advanced AI assistant with modular capabilities including networking, cryptography, web automation, AI integration, vision processing, security tools, and more.

## Features

- **Advanced Networking**: HTTP/2, WebSockets, MQTT, gRPC, network diagnostics
- **Advanced Cryptography**: AES, RSA, ECC, ECDSA, Zero-Knowledge Proofs, hashing
- **Web Operations**: Web scraping, search engines, API clients, content extraction
- **Automation**: Browser automation, file automation, system automation, RPA workflows
- **Database**: SQLite, JSON, Key-Value stores, Vector databases
- **AI Integration**: OpenAI, Anthropic, local LLMs, embeddings, AI agents
- **Vision**: Image processing, OCR, QR codes, face detection
- **Security**: Port scanning, SSL analysis, password strength, security headers
- **Voice**: Speech recognition, text-to-speech, wake word detection
- **Utilities**: Text processing, data conversion, file operations, math utilities

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python friday_assistant.py
```

## Modules

- `friday_assistant.py` - Main assistant integrating all modules

- `friday_voice.py` - Voice operations
- `friday_web.py` - Web operations
- `friday_automation.py` - Automation capabilities
- `friday_database.py` - Database operations
- `friday_ai.py` - AI integration
- `friday_tools.py` - Utility tools
- `friday_vision.py` - Vision processing
- `friday_security.py` - Security tools
- `advanced_networking.py` - Advanced networking
- `advanced_crypto.py` - Advanced cryptography
